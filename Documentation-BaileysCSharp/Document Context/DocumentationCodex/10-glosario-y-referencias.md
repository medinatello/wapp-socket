# 10. Glosario y referencias

## Glosario
- **JID**: Jabber ID, identificador de usuario/chat en WhatsApp.
- **PreKey**: Clave efímera utilizada en el protocolo Signal para iniciar sesiones.
- **WAM**: WhatsApp Web Message, formato binario usado por el servidor.
- **Node**: Estructura binaria similar a XML usada en el protocolo WA.
- **MEX**: API para newsletters de WhatsApp.
