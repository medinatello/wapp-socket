# 10. Glosario y referencias

## Glosario

| Término            | Definición |
|--------------------|-----------|
| **Baileys**        | Biblioteca TypeScript para interactuar con WhatsApp Web mediante WebSocket. |
| **LibSignal**      | Implementación del protocolo de cifrado Signal, usado para la mensajería en WhatsApp. |
| **Protobuf**       | Google Protocol Buffers, un formato eficiente para serializar datos estructurados. |
| **WebSocket**      | Protocolo que permite comunicaciones bidireccionales full‑duplex sobre una sola conexión TCP. |
| **Backoff**        | Técnica de reintento donde el tiempo de espera se incrementa (exponencial o lineal) después de cada fallo. |
| **Hexagonal Architecture** | Estilo arquitectónico que separa la lógica de negocio de las entradas y salidas mediante puertos y adaptadores. |
| **OpenTelemetry**  | Conjunto de APIs, SDKs y herramientas para instrumentar, generar, recopilar y exportar datos de observabilidad. |
